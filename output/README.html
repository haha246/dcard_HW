GitHub Rest API (Homework for Dcard frontend intern)

  1. Clone this project.
git clone https://github.com/haha246/dcard_HW
  1. Change the current directory.
cd dcard_HW
  1. Use yarn or npm to install all modules needed in this project.
yarn install

OR

npm install
  1. After installation...
yarn start

OR

npm start

  1. It should open your browser in a while, or you can open it by your own.
    http://localhost:3000 (By default.)

  2. Enjoy it!
